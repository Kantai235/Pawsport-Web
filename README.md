# Pawsport Web

Pawsport 的 GitHub Pages 介紹網站、隱私權政策與支援頁面。

## 網址

- 首頁：`https://kantai235.github.io/Pawsport-Web/`
- 隱私權政策：`https://kantai235.github.io/Pawsport-Web/privacy.html`
- 支援與聯絡：`https://kantai235.github.io/Pawsport-Web/support.html`

網站為純靜態 HTML/CSS，推送至 `main` 後由 GitHub Actions 部署到 GitHub Pages。

## App Store Connect

請將以下網址填入 App Store Connect：

- Privacy Policy URL：`https://kantai235.github.io/Pawsport-Web/privacy.html`
- Support URL：`https://kantai235.github.io/Pawsport-Web/support.html`
- Marketing URL（選填）：`https://kantai235.github.io/Pawsport-Web/`

App 正式上架後，請在首頁將「即將於 App Store 推出」替換為正式的 App Store 下載連結。
