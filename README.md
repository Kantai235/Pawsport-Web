# Pawsport Web

Pawsport 的 GitHub Pages 介紹網站、隱私權政策與支援頁面。

## 網址

- 首頁：`https://pawsport.init.engineer/`
- 隱私權政策（繁體中文）：`https://pawsport.init.engineer/privacy.html`
- Privacy Policy (English): `https://pawsport.init.engineer/privacy-en.html`
- 支援與聯絡：`https://pawsport.init.engineer/support.html`

網站為純靜態 HTML/CSS，推送至 `main` 後由 GitHub Actions 部署到 GitHub Pages。

## App Store Connect

請將以下網址填入 App Store Connect：

- Privacy Policy URL：`https://pawsport.init.engineer/privacy.html`
- Support URL：`https://pawsport.init.engineer/support.html`
- Marketing URL（選填）：`https://pawsport.init.engineer/`

App 正式上架後，請在首頁將「即將於 App Store 推出」替換為正式的 App Store 下載連結。
